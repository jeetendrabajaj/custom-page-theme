Add Custom page theme

To create a new custom page theme folow some simple rules.
1. Define the title of the theme.
2. A WordPress theme needs atleast two files to be exist - style.css and index.php
3. Be Sure:
   a. index.php file must must not contain the header and footer markups, So remove the html before start body tag <body> and after end body tag </body>.
   b. In index.php file no need to link the CSS or JavaSrcript files. Just upload the CSS or JavaSrcript files & Let the plugin enqueue CSS or JavaSrcript files in the defined order.
